# BFS (너비 우선 탐색)

## 1. 언제 떠올려야 하는가?
- 가중치 없는 그래프에서의 **최단 거리** 구하기
- 연결 요소(Connected Component) 찾기 (플러드 필)
- 상태 공간 트리에서 최소 이동 횟수

## 2. 사고 흐름 / 템플릿
1. `Queue` 생성 및 시작점 삽입
2. `visited` 배열 체크 (중복 방문 방지)
3. `while(queue is not empty)` 반복
4. 큐에서 꺼낸 노드의 인접 노드 탐색 (`dx`, `dy` 방향 배열 활용)
5. 방문하지 않은 인접 노드를 큐에 넣고 `visited = true` 처리

## 3. 자주 하는 실수
- **방문 체크 시점**: 큐에서 뺄 때가 아니라 **큐에 넣을 때** 체크해야 중복 방지됨
- `visited` 초기화 누락 (여러 테스트 케이스/여러번 탐색 시)
- 범위 체크(`x < 0`, `y >= N` 등)를 먼저 하지 않아 인덱스 에러
