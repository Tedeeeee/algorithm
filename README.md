# 🧠 Algorithm Study (Coding Test)

코딩테스트 대비를 위한 알고리즘 학습 및 문제 풀이 기록 저장소입니다.  
문제 풀이보다 **문제 구조를 이해하고, 알고리즘을 선택하는 사고 과정**을 남기는 것을 목표로 합니다.

---

## 🎯 학습 목표
- 실버 안정화 → 골드 초입 진입
- 문제를 읽자마자 알고리즘 유형을 분류할 수 있도록 훈련
- 같은 실수를 반복하지 않도록 기록으로 관리

---

---

## 📝 문제 풀이 기록 규칙
모든 문제는 아래 3가지를 반드시 기록합니다.

- **왜 이 알고리즘을 선택했는지** (한 줄)
- **시간복잡도**
- **실수 포인트 1개**

> 장황한 설명은 지양하고,  
> 코딩테스트 직전에 빠르게 복습할 수 있도록 정리합니다.

---

## 📌 정리 방식
- 알고리즘 개념 정리는 `docs/` 디렉터리에 유형별로 관리
- 문제별 긴 설명은 작성하지 않음
- 문제 기록은 **유형 문서에 누적**하여 관리
- 메리 크리스마스
